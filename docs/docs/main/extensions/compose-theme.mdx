# Compose Theme
There is a extension to change the Compose Theme.

## Setup
Add the `katalog-ext-theme` package.

```kotlin
dependencies {
    implementation("jp.co.cyberagent.katalog:katalog:`LATEST_VERSION`")
    implementation("jp.co.cyberagent.katalog:katalog-ext-theme:`LATEST_VERSION`")
}
```

You can set the Material Theme when calling `registerKatalog`.
The `background` color will be applied automatically.
Don't forget to set the `content`.

```kotlin {4-12}
registerKatalog(
    title = "Android Sample",
    extensions = listOf(
        ThemeExt { content ->
            MaterialTheme(
                colors = MaterialTheme.colors.copy(
                    background = Color.Red
                ),
                content = content
            )
        }
    )
) {
    compose("Text") {
        Text(text = "Hello, World")
    }
}
```
