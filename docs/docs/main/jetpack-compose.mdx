# Jetpack Compose

## Examples

<Preview
    imageUrl={useBaseUrl('/img/docs/jetpack-compose.png')}
    width={1080}
    height={2280} >

To add a `Composable`, use the `compose` method.

```kotlin
compose("Text") {
    Text(text = "Hello, World")
}
```

</Preview>

<Preview
    imageUrl={useBaseUrl('/img/docs/jetpack-compose-state.gif')}
    width={1080}
    height={2280} >

You can use state in the `compose`.

```kotlin
compose("MyComponent") {
    var boolean by remember {
        mutableStateOf(true)
    }
    Column(
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        Text(
            text = if (boolean) "Hello" else "World"
        )
        Button(
            onClick = { boolean = !boolean }
        ) {
            Text(text = "Toggle")
        }
    }
}
```

</Preview>

## Parameters

name|description
:--|:--
name|The UI Component name.
definition|Composable function.
